This is a project that uses a breadboard to cycle a 7segment LED through each number 0-9 and uses a button press and hold to cycle through the numbers.
If the player stops the LED on 0 the speed will increase. There's a few levels currently. 
The game tracks the number of presses it takes to complete all levels and tracks it on a highscore sheet.

I want to add a total time taken, a GUI, and a user list to enable personal high scores in both categories as well as a global highscore list.
